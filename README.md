# BOTS OPEN BY TAGS PARA MABILIS  

# 1. Blue Boxes = Momentum / RSI Trigger Zones
Yung mga blue blocks sa taas ng candles:
ibig sabihin may strong RSI condition na nangyari
parang “institution attention zone”
kadalasan lumalabas bago:
reversal
continuation
exhaustion
Napansin ko:
noong uptrend (May–Oct) → lumalabas sila habang malakas trend
noong downtrend (Nov–Feb) → halos wala or delayed
ngayon (Apr–May) → bumabalik ulit sila
Meaning:
sinusubukan ng market bumuo ng bagong bullish structure.

# 2. Background Colors (Red/Green)
Ito pinaka-interesting part.
Green Zones
Mukhang:
bullish RSI environment
accumulation phase
momentum recovery
Red Zones
Mukhang:
bearish pressure
distribution

Ang astig:
Hindi siya random.

makikita mo mood ng BTC.

# 3. EMA Flow
May dalawang blue lines:
fast EMA
slow EMA
Kapag:
fast nasa ibabaw = bullish flow
fast nasa ilalim = bearish flow
Ngayon sa current area:
medyo nagco-converge
parang nagdadalawang isip market
Kaya tama yung nasa kanan:
Trend: NEUTRAL
Signal: WAIT
Actually mature signal yan.
Hindi pilit bullish.
Hindi pilit bearish.
Professional systems madalas:
“wait first before attack.”
